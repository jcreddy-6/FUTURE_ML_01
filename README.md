## 📌 Objective
Build a machine learning model to forecast future sales using historical
business data, providing actionable business insights through visualizations.

---

## 📂 Dataset
- **Source:** [Kaggle - Sales Forecasting Dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting)
- **Records Analysed:** 1199 daily sales records
- **Date Range:** 2015-02-14 → 2018-12-30

---

## 🛠️ Tech Stack
| Tool | Purpose |
|------|---------|
| Python 3.10+ | Core language |
| Pandas | Data cleaning & manipulation |
| NumPy | Numerical computations |
| Matplotlib / Seaborn | Data visualization |
| Scikit-learn | ML model training & evaluation |

---

## 🔑 Key Features
- ✅ Data cleaning & preprocessing
- ✅ Time-based feature engineering (lag features, rolling means)
- ✅ Forecasting using Linear Regression & Random Forest
- ✅ Model evaluation with MAE, RMSE, MAPE
- ✅ Business-friendly forecast visualizations
- ✅ Feature importance analysis

## 🚀 How to Run
1. Clone the repository
```bash
git clone https://github.com/jcreddy-6/FUTURE_ML_01.git
cd FUTURE_ML_01
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Open the notebook
```bash
code sales_forecasting.ipynb
```

4. Run all cells (`Ctrl + Shift + P` → Run All)
